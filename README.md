# Gestor de Incidencias
## Introducción

Este proyecto es una aplicación web dinámica desarrollada en PHP nativo (sin frameworks) que implementa un sistema básico de Gestión de Incidencias (Helpdesk).

El objetivo es construir una aplicación CRUD (Crear, Leer, Actualizar, Borrar) completa y segura que interactúe con una base de datos MySQL. La aplicación permite gestionar tickets de incidencias, registrando su título, descripción, prioridad y estado (abierta/cerrada).

## 1. Clonamos repositorio y accedemos
```bash
git clone https://github.com/4ntoniomj/PHP
````
